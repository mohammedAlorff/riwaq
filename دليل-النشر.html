🚀 دليل نشر موقع رواق

هذا الدليل يأخذك خطوة بخطوة لرفع موقع رواق على InfinityFree (مجاني 100%).

⏱️ الوقت المتوقع: 15–20 دقيقة | لا تحتاج خبرة تقنية متقدمة

الخطوة 1 — إنشاء حساب InfinityFree

1 افتح infinityfree.com واضغط Sign Up Free

2 سجّل بإيميلك وتأكّد منه

3 ادخل للوحة التحكم واضغط Create Account لإنشاء موقع جديد

الخطوة 2 — إنشاء قاعدة البيانات

1 من لوحة التحكم، افتح MySQL Databases

2 أنشئ قاعدة بيانات جديدة — ستحصل على بيانات مثل:
Host: sql123.epizy.com
Database: epiz_12345678_riwaq
Username: epiz_12345678
Password: (كلمة المرور اللي اخترتها)
3 احتفظ بهذه البيانات — ستحتاجها في الخطوة التالية

الخطوة 3 — ضبط config.php

افتح ملف api/config.php وعدّل القيم بالبيانات اللي أخذتها:
'host' => 'sql123.epizy.com', // ← Host من InfinityFree
'name' => 'epiz_12345678_riwaq', // ← اسم قاعدة البيانات
'user' => 'epiz_12345678', // ← اسم المستخدم
'password' => 'كلمة-مرورك', // ← كلمة المرور

الخطوة 4 — رفع الملفات

1 من لوحة InfinityFree، افتح File Manager

2 انتقل لمجلد htdocs

3 ارفع ملف riwaq-deploy.zip (موجود عندك في مجلد المشروع)

4 بعد الرفع، انقر بزر اليمين على الملف واختر Extract

5 انقل محتويات مجلد riwaq-deploy مباشرة إلى htdocs

الخطوة 5 — تهيئة قاعدة البيانات

افتح المتصفح وانتقل لـ:
https://your-site.epizy.com/api/setup.php
يجب أن ترى رسالة نجاح تقول "تم إنشاء الجداول". هذا يُنشئ الجداول ويضيف البيانات الأولية.
⚠️ مهم: بعد التهيئة، احذف أو أعد تسمية ملف api/setup.php لأسباب أمنية.
